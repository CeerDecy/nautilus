| 指令                           | 参数                          | 说明        | 
|------------------------------|-----------------------------|-----------|
| list task                    | 无                           | 列出当前所有的任务 | 
| create task {name} {context} | name：任务名称<br/>context：任务上下文 | 创建出排查任务   |
| none {context}               | context：可选，输出不执行指令的原因       | 不需要执行指令   |
